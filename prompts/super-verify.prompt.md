# /super-verify

Use este alias quando quiser acionar a verificação final obrigatória com o nome curto do plugin original.

## Equivalência

- comando compatível: `/super-verify`
- prompt canônico no mirror Copilot: `/superpowers-verification-before-completion`

## O que fazer

1. Revisar plano aprovado, batches executados e validações disponíveis.
2. Apresentar findings primeiro, resumo depois.
3. Declarar `pass`, `warn` ou `fail`.
4. Encerrar com próximo passo corretivo quando o veredito não for `pass`.

## Transparência

- sinalizar que este comando é um alias de compatibilidade;
- manter a saída operacional alinhada ao prompt canônico `/superpowers-verification-before-completion`.
