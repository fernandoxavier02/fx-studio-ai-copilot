# /super-brainstorm

Use este alias quando quiser abrir a etapa de brainstorming com o nome curto do plugin original.

## Equivalência

- comando compatível: `/super-brainstorm`
- prompt canônico no mirror Copilot: `/superpowers-brainstorming`

## O que fazer

1. Executar a mesma disciplina de exploração do prompt canônico.
2. Delimitar objetivo, restrições, riscos, fatos verificados e lacunas.
3. Encerrar com handoff explícito para `/superpowers-writing-plans` quando a exploração estiver madura.

## Transparência

- sinalizar que este comando é um alias de compatibilidade;
- manter a saída operacional alinhada ao prompt canônico `/superpowers-brainstorming`.
